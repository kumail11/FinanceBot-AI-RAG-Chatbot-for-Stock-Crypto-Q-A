# FinanceBot-AI-RAG-Chatbot-for-Stock-Crypto-Q-A
AI-powered RAG chatbot for real-time stock &amp; crypto Q&amp;A — built with FastAPI, LangChain, ChromaDB, and free APIs (CoinGecko, Alpha Vantage, Finnhub, OpenRouter). Ask anything about markets in natural language!

An intelligent, real-time financial assistant powered by **Retrieval-Augmented Generation (RAG)** that answers natural language questions about stocks, cryptocurrencies, market news, and financial concepts — all using **100% free APIs**.

Ask: *"What's Bitcoin's price?"* · *"Tesla ka P/E ratio kya hai?"* · *"Latest AAPL news"* · *"Should I invest in ETH?"*

Key Features

- **Natural Language Q&A** — Ask financial questions in English or Roman Urdu
- **Real-Time Data** — Live crypto prices, stock quotes & market news
- **RAG Architecture** — LangChain + ChromaDB knowledge retrieval for smarter answers
- **Sentiment Analysis** — News headlines auto-tagged as Bullish/Bearish/Neutral (VADER)
- **Conversation Memory** — Context-aware multi-turn chat sessions
- **100% Free APIs** — CoinGecko, Alpha Vantage, Finnhub, OpenRouter/Groq
- **Modern Chat UI** — Clean, responsive dark-theme frontend

- Tech Stack

| Layer           | Technology                                     |
|-----------------|------------------------------------------------|
| Backend         | FastAPI, Python                                |
| AI/RAG Pipeline | LangChain, ChromaDB, HuggingFace Embeddings    |
| LLM             | OpenRouter (Llama 3.3 70B Free) / Groq         |
| Data Sources    | CoinGecko, Alpha Vantage, Finnhub              |
| Sentiment       | VADER (Local, No API needed)                   |
| Frontend        | Vanilla HTML/CSS/JS (Dark Theme)               |
| Deployment      | Docker, Uvicorn                                |
